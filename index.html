This page has a <canvas> element within a 16px by 16px <div> located between these two paragraphs. The JavaScript (external, loaded at the end of the <body>) takes an array of color values and renders them pixel-by-pixel to the canvas.

Select a sprite below to load into the canvas. The images used below are just plain PNG files, but the sprites loaded into the canvas are all arrays of RGB values. Each sprite has different pixel dimensions, and the script resizes the canvas element to match the sprite loaded.
